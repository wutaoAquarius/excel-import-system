## 一、命名需求分析

### 1.1 系统核心特性
- ✅ **多模板识别**: 自动识别不同格式的Excel模板
- ✅ **自学习**: 模板映射规则自动学习和记忆
- ✅ **批量导入**: 支持1000+行数据的无缝导入
- ✅ **智能匹配**: 列名相似度算法自动映射
- ✅ **在线编辑**: 导入后支持在线编辑和校验
- ✅ **一键下单**: 完整的批量下单工作流

### 1.2 目标用户
- 物流/快递行业从业人员
- 供应链管理人员
- 批量数据录入工作者
- 企业管理系统集成人员

### 1.3 命名原则
```
P1 - 品牌性    | 易记、有辨识度、专业
P2 - 功能性    | 体现核心功能（导入、识别、学习）
P3 - 国际化    | 支持中英文，便于推广
P4 - 域名可用  | 对应.com域名可注册
P5 - SEO友好   | 便于搜索优化
P6 - 视觉化    | 易于设计Logo
```

## 七、品牌故事和宣传语

### 7.1 品牌故事

```
ImportWise的起源:

在物流行业的某个繁忙的午后，
一位供应链经理面临着一个困境：

来自10个不同供应商的订单数据，
都是Excel格式，但格式五花八门：
- 有的用"收件人"，有的用"收方"
- 有的列序不同，有的还有合并单元格
- 每次导入都要手工调整，浪费了大量时间

她说："为什么没有一个工具，
能像个聪明人一样，
自动理解这些不同的格式呢？"

ImportWise就是这样诞生的。

我们相信：
⭐ 数据导入，应该是聪明的
⭐ 系统应该学会适应，而不是让用户适应
⭐ 智慧的工具，应该被所有人使用

ImportWise - 一次学习，永久导入。
```
