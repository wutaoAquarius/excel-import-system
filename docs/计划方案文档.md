## 功能概述

开发一个Web应用，支持物流/快递行业的多种Excel模板自动导入下单。核心能力包括：
- 支持至少5种不同格式的Excel模板自动识别与映射
- 实现模板记忆学习功能，一次调整永久记住映射规则
- 支持1000+条数据的无缝导入，性能优化体验流畅
- 完整的数据校验与错误提示，一次性全量展示所有问题
- 提供在线编辑、导出、历史查看等完整功能

```sql
  id SERIAL PRIMARY KEY,

  sender_name VARCHAR(100) NOT NULL,

  sender_address TEXT NOT NULL,

  total_count INT NOT NULL,
  success_count INT DEFAULT 0,
  failed_count INT DEFAULT 0,
  status VARCHAR(50) DEFAULT 'processing',
  error_details JSONB,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  completed_at TIMESTAMP
);

-- 创建索引
CREATE INDEX IF NOT EXISTS idx_orders_external_code ON orders(external_code);
CREATE INDEX IF NOT EXISTS idx_orders_batch_number ON orders(batch_number);

```

### 核心算法设计

#### 3.1 相似度匹配算法
```typescript
// 策略：多层级匹配
1. 精确匹配（直接对应）
2. 规则匹配（规则库）
3. 相似度匹配（Levenshtein）
4. 语义匹配（同义词识别）

// 优先级：规则库 > 精确匹配 > 相似度匹配
```

- **规则库映射**（中英混合）：
  ```typescript
  const columnAliasMap = {
    '发件人姓名': ['发件人', '寄件人', '寄件人姓名', 'Sender Name', 'sender_name'],
    '发件人电话': ['发件人电话', '寄件人电话', '电话', 'Sender Phone', 'sender_phone'],
    '收件人姓名': ['收件人', '收货人', '收方', 'Receiver Name', 'receiver_name'],
    '收件人电话': ['收件人电话', '收货人电话', '联系电话', 'Receiver Phone', 'receiver_phone'],
    '收件人地址': ['收件人地址', '收货地址', '完整地址', 'Address', 'receiver_address'],
    // ... 更多别名
  };
  ```

- **Levenshtein距离计算**：
  - 阈值设定：相似度 > 0.6 认为匹配
  - 支持中英文字符权重不同

#### 3.2 模板指纹生成
```typescript
// 指纹 = 表头列名集合的哈希
function generateTemplateFingerprint(headers: string[]): string {
  // 将表头按字母顺序排序，去除重复
  const sorted = Array.from(new Set(headers)).sort();
  // 生成SHA256哈希
  return sha256(sorted.join('|'));
}
```

#### 3.3 模板学习存储流程
```
用户上传Excel
  → 提取表头
  → 生成指纹
  → 查询历史映射 (template_mappings 表)
  → 找到相同/相似指纹？
    ├─ 是 → 自动应用映射，进入预览编辑
    └─ 否 → 进入手动映射流程
  → 用户确认映射后
    → 保存映射规则到数据库
    → 后续相同表头自动应用
```

#### 5.2 验证函数
```typescript
function validateRow(row: any, rules: ValidationRule[]): ValidationError[] {
  const errors = [];

  for (const rule of rules) {
    const value = row[rule.field];

    // 必填检查
    if (rule.required && !value) {
      errors.push({
        field: rule.field,
        type: 'required',
        message: `${rule.field}：缺失必填字段`
      });
      continue;
    }

    // 格式检查
    if (value && rule.pattern) {
      if (!rule.pattern.test(String(value))) {
        errors.push({
          field: rule.field,
          type: 'format',
          message: `${rule.field}：格式错误`
        });
      }
    }

    // 类型检查
    if (value && rule.type === 'number') {
      if (isNaN(Number(value)) || Number(value) <= 0) {
        errors.push({
          field: rule.field,
          type: 'type',
          message: `${rule.field}：${rule.message}`
        });
      }
    }

    // 枚举检查
    if (value && rule.enum) {
      if (!rule.enum.includes(value)) {
        errors.push({
          field: rule.field,
          type: 'enum',
          message: `${rule.field}：不在可选范围内`
        });
      }
    }
  }

  return errors;
}
```
